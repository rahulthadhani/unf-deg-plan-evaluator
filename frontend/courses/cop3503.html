COP 3503 — Programming II

3 Credit Hours  ·  BS Computer Science Requirement  · 
COP 3503 · 3 Credit HoursProgramming II

This course extends the fundamentals of programming introduced in Programming I. Students explore object-oriented programming concepts including classes, inheritance, polymorphism, and encapsulation. Topics include recursion, algorithm analysis, and an introduction to data structures such as linked lists, stacks, queues, and trees.
