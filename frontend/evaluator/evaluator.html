Usability Evaluation

Task-based evaluation of the UNF Degree Planner interface

Welcome to the Evaluation

This session guides you through a series of realistic tasks using the UNF Degree Planner. Each task presents a scenario you might encounter as a student managing your coursework.

Before each task begins you will see a description of the scenario. A timer starts once you click Begin Task. Complete each task as quickly and accurately as you can. A Quit Task button is always available if you are unable to complete a task.

Note: If you have previously used the planner, click Reset All Progress below to start from a clean state.

Evaluation Tasks
Participant # — edit to set number; increments automatically on each reset

Completed task records and app data are stored in your browser. Reset clears all evaluation progress and enrolled course data so you can start fresh.
